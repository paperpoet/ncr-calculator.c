# nCr Calculator in C

This C program calculates combinations (nCr) using the factorial formula:

> nCr = n! / (r! * (n - r)!)

## How to Run

1. Compile the code:
   ```bash
   gcc ncr.c -o ncr
./ncr

2. Run the program
  ./ncr
